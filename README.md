# SJTU-Classroom
SJTU teaching building information release system, 2022 SJTU-ICE2604-Introduction to Software Engineering
* Frame: VUE2 
* Back-end: FastAPI + node express 
* Search engine: elasticsearch + logstash + sql 
* Visualization: echarts
