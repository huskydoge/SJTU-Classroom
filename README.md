# SJTU-Classroom
SJTU教学楼信息发布系统，2022 ICE电类工程导论开放大作业自选项目

* 框架：VUE2
* 后端：FastAPI + node express
* 搜索引擎：elasticsearch + logstash + sql
* 可视化：echarts
